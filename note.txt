ProfileImageSelector component 

1. **Component Purpose**: This React component (`ProfilePhotoSelector`) lets users upload, preview, and remove a profile photo.  
2. **Hooks Used**: `useRef` for file input access and `useState` to track the image file and preview URL.  
3. **File Handling**: When a file is selected (`handleImageChange`), it updates the image state and generates a preview URL.  
4. **Remove Function**: `handleRemoveImage` clears the selected image and preview.  
5. **UI Logic**: Shows a default user icon when no image is selected, or the preview if an image exists.  
6. **Interactive Elements**: Clicking the upload button (LuUpload) triggers the file input; the trash icon (LuTrash) removes the image.  
7. **Styling**: Uses Tailwind CSS for responsive styling, rounded avatars, and positioned action buttons.
