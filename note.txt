Download All Users Task Report.

Explaining the handleDownloadReport functionality
1. **Function Purpose**: Downloads a user report as an Excel file from the backend.  
2. **API Call**: Uses `axiosInstance` to fetch data from `EXPORT_USERS` endpoint with `responseType: "blob"` for binary data.  
3. **Blob Handling**: Converts the response into a Blob (binary object) for file creation.  
4. **URL Creation**: Generates a temporary URL for the Blob using `window.URL.createObjectURL()`.  
5. **Link Setup**: Creates an invisible `<a>` tag to trigger the download programmatically.  
6. **File Naming**: Sets the filename to `users_report.xlsx` via the `download` attribute.  
7. **DOM Manipulation**: Temporarily adds the link to the document body to enable clicking.  
8. **Trigger Download**: Simulates a click on the link to start the download.  
9. **Cleanup**: Removes the link and revokes the Blob URL to free memory.  
10. **Error Handling**: Logs errors and shows a toast notification if the download fails.
