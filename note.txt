SeETTING UP MONGODB ATLAS.

🧰 What is MongoDB Atlas?
MongoDB Atlas is a cloud-based MongoDB service — so you don’t need to install MongoDB on your local machine. It gives you a secure, scalable, online database.

⚙️ How to Set It Up (Step-by-Step)
✅ 1. Create a MongoDB Atlas Account
Go to: https://www.mongodb.com/cloud/atlas
Sign up (it's free for basic usage)

✅ 2. Create a New Project & Cluster
Click "New Project" → Give it a name
Inside the project, click "Build a Database"
Choose Shared (Free) tier
Pick a Cloud Provider + Region (e.g., AWS + your nearest region)
Name your Cluster (optional) and click "Create Cluster"

✅ 3. Create a Database User
Go to Database Access → Click “Add New Database User”
Set:
Username + Password
Role: Keep default (readWriteAnyDatabase)
Click Create User

✅ 4. Allow Your IP Address
Go to Network Access → Click "Add IP Address"
Choose:
“Allow Access from Anywhere” (0.0.0.0/0) for dev use
Or enter your current IP
Click Confirm

✅ 5. Get Your Connection URI
Go to Clusters → Click "Connect"
Choose “Connect Your Application”
Copy the connection string, e.g.:
mongodb+srv://<username>:<password>@cluster0.abcd.mongodb.net/loan-app?retryWrites=true&w=majority
Replace <username> and <password> with the ones you created.

✅ 6. Use It in Your App
In your .env file:
MONGO_URI=mongodb+srv://yourUser:yourPassword@cluster0.abcd.mongodb.net/loan-app?retryWrites=true&w=majority
